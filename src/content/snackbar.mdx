# Snackbar

The snackbar is a horizontal bar with the following CSS used inside of a toast (regen the CSS to match the requirements. This is the default CSS. Replace variable sizes like “sm” with their liftkit size unit equivalents. replace instances of body-xs with the result of multiplying line height of the body element by halfstep-dec. Replace instances of callout-sm with result of multiplying line height of callout by wholestep-dec. Replace colors with the correct colors and use shadow —xs variable instead of the explicit values.

```css
display: inline-flex;
padding: var(--body-xs, 7.045px);
align-items: center;
gap: sm;
border-radius: var(--callout-sm, 11.804px);
background: var(--LiftKit-Default-sys-light-surface, #FBF8FF);

/* shadow-xs */
box-shadow: 0px 0px 1px 0px #7A757F, 0px 2px 4px 0px rgba(63, 73, 71, 0.10), 0px 4px 6px -1px rgba(63, 73, 71, 0.15);
```

The content of a snackbar element is a badge component, a Text component, and 2 button components: an outlined and a filled, both with size=”sm.”

**Props**

- badgeColor
    - Sets color prop of the badge to the passed value.
- primaryButtonColor
    - Sets color prop of the filled button to the passed value.
- secondaryButtonColor
    - Sets color prop of the outlined button to the passed value.
- backgroundColor
    - Sets the bg color of the snackbar itself to the passed value and the text color to the equivalent “on” token.
- globalColor
    - Sets bg color of snackbar to the passed value’s CONTAINER equivalent, the text color to the ON CONTAINER token, and the badge and button colors to the passed value.
