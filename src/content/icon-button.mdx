# Icon Button

## Base Component

**Default Behavior**

The `IconButton` is a `<button>` element that wraps a single `Icon` component. It follows the same layout and structure as the `Button` component but omits any label or trailing icon. The component scales based on its `size` prop, which controls the font size of the button—this implicitly scales all child units since they’re defined in `em` units.

### HTML

```html
<button
  lk-component="icon-button"
  lk-icon-button-size="md"
  lk-icon-button-variant="fill"
  lk-icon-button-color="lk-primary"
  type="button"
>
  <div lk-button-content-wrap>
    <i lk-component="icon">favorite</i>
  </div>
  <div lk-component="state-layer"></div>
</button>
```

### TypeScript

```tsx
import { useMemo } from "react";
import { propsToDataAttrs } from "../utilities";

type LkSizeUnit = "xs" | "sm" | "md" | "lg" | "xl";

interface LkIconButtonProps extends React.ButtonHTMLAttributes<HTMLButtonElement> {
  icon: string;
  variant?: "fill" | "outline" | "text";
  color?: string; // LkColor
  size?: LkSizeUnit;
}

export default function IconButton({
  icon,
  variant = "fill",
  color = "lk-primary",
  size = "md",
  ...rest
}: LkIconButtonProps) {
  const dataAttrs = useMemo(() => propsToDataAttrs({ variant, color, size }, "lk-icon-button"), [variant, color, size]);

  return (
    <button type="button" {...dataAttrs} {...rest}>
      <div lk-button-content-wrap>
        <i lk-component="icon">{icon}</i>
      </div>
      <div lk-component="state-layer"></div>
    </button>
  );
}
```

```css

  display: inline-block;
  position: relative;
  border-radius: 100em;

  position: absolute;
  inset: 0;
  pointer-events: none;

  opacity: 0;
  transition: background-color 0.2s ease-out, opacity 0.2s ease-out;

[lk-component="icon-button"]:hover > [lk-component="state-layer"],
[lk-component="icon-button"]:focus-visible > [lk-component="state-layer"] {
  opacity: 0.04;
}

/* Size map */
[lk-icon-button-size="xs"] {
  font-size: var(--caption-font-size);
}

[lk-icon-button-size="sm"] {
  font-size: var(--caption-line-height);
}

[lk-icon-button-size="md"] {
  font-size: var(--heading-font-size);
}

[lk-icon-button-size="lg"] {
  font-size: var(--title3-font-size);
}

[lk-icon-button-size="xl"] {
  font-size: var(--title2-font-size);
}
```

## Combine with Native HTML Attributes

You can set any other _native_ HTML attribute (className, style, id, etc.) on the component by passing it as a prop.

## Props

### **icon**

**Description**

Sets the Material Symbol name used inside the Icon.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| ----------------- | ----------------- | --------------------- |
| `string`          | _(none)_          | Any Material Symbol   |

```tsx
<IconButton icon="favorite" />
```

### **variant**

**Description**

Controls the button's visual style—solid fill, outline, or text only.

| **Expected Type** | **Default Value** | **Acceptable Values**     |
| ----------------- | ----------------- | ------------------------- |
| `string`          | `fill`            | `fill`, `outline`, `text` |

```tsx
<IconButton variant="outline" />
```

### **color**

**Description**

Sets the color token applied to the state layer and icon color.

| **Expected Type**  | **Default Value** | **Acceptable Values**   |
| ------------------ | ----------------- | ----------------------- |
| `string` (LkColor) | `lk-primary`      | Any LiftKit color token |

```tsx
<IconButton color="lk-error" />
```

### **size**

**Description**

Controls the component's font size, which scales all child units proportionally.

| **Expected Type** | **Default Value** | **Acceptable Values**        |
| ----------------- | ----------------- | ---------------------------- |
| `LkSizeUnit`      | `md`              | `xs`, `sm`, `md`, `lg`, `xl` |

```tsx
<IconButton size="lg" />
```
