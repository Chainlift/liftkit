# Image

## Base Component

**Default Behavior**

The `Image` component is a simple `<img>` element that allows configuration of display properties like aspect ratio, border radius, object-fit, and dimensions via props. It is primarily used for non-background images within the layout system.

### HTML

```html
<img
  lk-component="image"
  lk-image-aspect="auto"
  lk-image-fit="fill"
  lk-image-radius="none"
  lk-image-width="auto"
  lk-image-height="auto"
  src="path/to/image.jpg"
  alt="Descriptive text"
/>

```

### TypeScript

```tsx
import { useMemo } from 'react';
import { propsToDataAttrs } from './utilities';

type LkAspectRatio = 'auto' | '1/1' | '2.39/1' | '2/1' | '16/9' | '3/2' | '4/3' | '5/4' | '1/2.39' | '1/2' | '9/16' | '4/5';
type LkSizeUnit = '3xs' | '2xs' | 'xs' | 'sm' | 'md' | 'lg' | 'xl' | '2xl' | '3xl' | '4xl';

type LkImageProps = React.ImgHTMLAttributes<HTMLImageElement> & {
  aspect?: LkAspectRatio;
  borderRadius?: LkSizeUnit | 'none' | 'zero' | undefined;
  objectFit?: React.CSSProperties['objectFit'];
  width?: LkSizeUnit | 'auto';
  height?: LkSizeUnit | 'auto';
};

export default function Image({ aspect = 'auto', borderRadius = undefined, objectFit = 'fill', width = 'auto', height = 'auto', ...rest }: LkImageProps) {
  const attrs = useMemo(() => propsToDataAttrs({ aspect, borderRadius, objectFit, width, height }, 'lk-image'), [aspect, borderRadius, objectFit, width, height]);

  return <img {...attrs} {...rest} />;
}

```

### CSS (Reference)

```css
[lk-component="image"] {
  display: block;
  width: 100%;
  height: auto;
  object-fit: var(--lk-image-fit, fill);
}

[lk-image-radius]:not([lk-image-radius="none"]):not([lk-image-radius="zero"]) {
  border-radius: var(--lk-size-md);
  overflow: hidden;
}

```

## Combine with Native HTML Attributes

You can set any other *native* HTML attribute (alt, src, loading, etc.) on the component by passing it as a prop.

## Props

### **aspect**

**Description**

Controls the aspect ratio of the image.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `LkAspectRatio` | `auto` | `auto`, `1/1`, `2.39/1`, `2/1`, `16/9`, `3/2`, `4/3`, `5/4`, `1/2.39`, `1/2`, `9/16`, `4/5` |

```tsx
<Image aspect="16/9" />

```

### **borderRadius**

**Description**

Controls the border radius of the image. When set, overflow is hidden.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `LkSizeUnit | 'none' | 'zero' | undefined` | `undefined` | Any size unit, `'none'`, `'zero'` |

```tsx
<Image borderRadius="lg" />

```

### **objectFit**

**Description**

Controls how the image should be resized to fit its container.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `objectFit` | `fill` | `'fill'`, `'cover'`, `'contain'`, etc. |

```tsx
<Image objectFit="cover" />

```

### **width**

**Description**

Controls the image width.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `LkSizeUnit | 'auto'` | `auto` | Size unit or `'auto'` |

```tsx
<Image width="lg" />

```

### **height**

**Description**

Controls the image height.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `LkSizeUnit | 'auto'` | `auto` | Size unit or `'auto'` |

```tsx
<Image height="md" />

```
