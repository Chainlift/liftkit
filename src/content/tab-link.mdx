# Tab Link

## Base Component

**Default Behavior**

The `TabLink` component is a horizontally oriented flex `div` with a `Text` component and an absolutely positioned 2px tall `div` that acts as the active tab indicator. The indicator appears at the bottom of the container. Its opacity is controlled via the `selected` prop.

The component accepts only one generic child: a single `<div>` element, which can internally contain other elements but cannot be a list or multiple siblings.

By default, all sides are padded with `var(--lk-size-md)`.

When active:

- The indicator has `opacity: 1`
- The text component is set to bold manually (e.g. `font-weight: 700`)

### HTML

```html
<div lk-component="tab-link" lk-tab-link-selected="false">
  <div lk-slot="child">
    <Text fontClass="body">Tab</Text>
  </div>
  <div class="lk-indicator"></div>
</div>

```

### TypeScript

```tsx
import { useMemo } from 'react';
import { propsToDataAttrs } from './utilities';

interface LkTabLinkProps extends React.HTMLAttributes<HTMLDivElement> {
  selected?: boolean;
  fontClass?: LkFontClass;
  indicatorClass?: string;
  children?: React.ReactElement<HTMLDivElement>;
}

export default function TabLink({
  selected = false,
  fontClass = 'body',
  indicatorClass = 'lk-indicator',
  children,
  ...rest
}: LkTabLinkProps) {
  const dataAttrs = useMemo(() => propsToDataAttrs({ selected }, 'lk-tab-link'), [selected]);

  return (
    <div {...dataAttrs} {...rest}>
      <div lk-slot="child">
        <Text fontClass={fontClass} style={selected ? { fontWeight: 700 } : {}}>
          {children}
        </Text>
      </div>
      <div className={indicatorClass}></div>
    </div>
  );
}

```

### CSS (Reference)

```css
[lk-component="tab-link"] {
  display: flex;
  position: relative;
  padding: var(--lk-size-md);
}

.lk-indicator {
  position: absolute;
  bottom: 0;
  left: 0;
  width: 100%;
  height: 2px;
  background-color: var(--lk-primary);
  opacity: 0;
  transition: opacity 0.2s ease-out;
}

[lk-tab-link-selected="true"] > .lk-indicator {
  opacity: 1;
}

```

## Combine with Native HTML Attributes

You can set any other *native* HTML attribute (className, style, id, etc.) on the component by passing it as a prop.

## Props

### **selected**

**Description**

Indicates whether the tab is currently active. Controls the indicator and text weight.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `boolean` | `false` | `true`, `false` |

```tsx
<TabLink selected={true}>Tab</TabLink>

```

### **fontClass**

**Description**

Sets the font size/style of the text inside the tab.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `LkFontClass` | `'body'` | Any LkFontClass value |

```tsx
<TabLink fontClass="title3">Tab</TabLink>

```

### **indicatorClass**

**Description**

Allows you to override the default styles of the indicator using a custom class.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `string` | `lk-indicator` | Any valid class name |

```tsx
<TabLink indicatorClass="my-indicator-style" />

```
