# Section

## Section Component

The Section is a `<section>` element whose sole purpose is controlling padding at the section level. It includes one slot designed specifically to hold a Container component.

### Default Behavior

By default, the Section component:

- Has `display: block`
- Has `position: relative`
- Includes padding equal to `var(--lk-size-2xl)` on all sides

### HTML Structure

```html
<section lk-component="section" lk-section-padding="md">
  <div lk-component="slot" lk-slot="container"></div>
</section>
```

### CSS Implementation

Path:

```
..someParentFolder/liftkit/components/section.css
```

Default CSS Rules:

```css
[lk-component="section"] {
  display: block;
  position: relative;
  padding: var(--lk-size-2xl);
}

/* Padding attribute variations */
[lk-section-padding="none"] { padding: 0; }
[lk-section-padding="xs"] { padding: var(--lk-size-xs); }
[lk-section-padding="sm"] { padding: var(--lk-size-sm); }
[lk-section-padding="md"] { padding: var(--lk-size-md); }
[lk-section-padding="lg"] { padding: var(--lk-size-lg); }
[lk-section-padding="xl"] { padding: var(--lk-size-xl); }
```

### React / TypeScript Implementation

### Interface

```tsx
import { useMemo } from 'react';
import { propsToDataAttrs } from './utilities';

interface LkSectionProps extends React.HTMLAttributes<HTMLElement> {
  padding?: 'xs' | 'sm' | 'md' | 'lg' | 'xl' | 'none';
  container?: React.ReactNode;
}

export default function Section(props: LkSectionProps) {
  const { container, children, ...restProps } = props;

  const lkSectionAttrs = useMemo(
    () => propsToDataAttrs(restProps, 'lk-section'),
    [restProps.padding]
  );

  return (
    <section {...lkSectionAttrs}>
      <div lk-component="slot" lk-slot="container">
        {container}
      </div>
      {children}
    </section>
  );
}
```

### Example Instance

```tsx
import Section from './Section';
import Container from './Container';

function ExampleSectionPage() {
  return (
    <Section padding="lg" className="my-custom-section" style={{ marginTop: '3rem' }} container={<Container />}>
      <p>Additional content here.</p>
    </Section>
  );
}
```

### Props

The Section component accepts the following props:

### `padding`

Controls padding on all sides of the section. To override padding on specific sides, apply a utility class via the `className` attribute.

| **Expected Type** | **Default Value** | **Acceptable Values** | **Implementation Example** |
| --- | --- | --- | --- |
| `string` | `md` | `xs`, `sm`, `md`, `lg`, `xl`, `none` | `<Section padding="sm" />` |

### `container` (slot)

Accepts any HTML element or React component, intended specifically for a Container component.

| **Expected Type** | **Default Value** | **Acceptable Values** | **Implementation Example** |
| --- | --- | --- | --- |
| `React.ReactNode` | None | Any HTML element or React component | `<Section container={<Container />} />` |
