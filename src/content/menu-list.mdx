# Menu List

## Base Component

**Default Behavior**

The `MenuList` component is an absolutely positioned dropdown container for `MenuItem` components or other valid React nodes. Its visibility is toggled using the `isOpen` prop. Internally, it uses a `Column` component that aligns children with `stretch` and defines spacing using LiftKit size tokens.

- **Position:** `absolute`
- **Box shadow:** `var(--shadow-xl)`
- **Padding:** `var(--lk-size-md)` on all sides
- **Border radius:** `var(--lk-size-md)`
- **Gap between items (inside column):** `calc(var(--lk-size-lg) * var(--lk-halfstep-dec))`
- **Width:** `var(--lk-size-3xl)`
- **Max height:** `var(--lk-size-4xl)` (overridable via prop)
- **Overflow:** `auto` (only when open)
- **Transition:** animated open/close via scale, opacity, and display rules

### HTML

```html
<div lk-component="menu-list" lk-menu-list-open="true">
  <div lk-component="column">
    <div lk-component="menu-item">Menu Item 1</div>
    <div lk-component="menu-item">Menu Item 2</div>
  </div>
</div>

```

### TypeScript

```tsx
import { useMemo } from 'react';
import { propsToDataAttrs } from './utilities';

interface LkMenuListProps extends React.HTMLAttributes<HTMLDivElement> {
  isOpen?: boolean;
  maxHeight?: string; // Accepts CSS-compatible values like 'auto' or LiftKit size vars
  children?: React.ReactNode;
}

export default function MenuList({ isOpen = false, maxHeight = 'var(--lk-size-4xl)', children, ...rest }: LkMenuListProps) {
  const dataAttrs = useMemo(() => propsToDataAttrs({ open: isOpen }, 'lk-menu-list'), [isOpen]);

  return (
    <div
      style={{
        maxHeight: isOpen ? maxHeight : '0px',
        display: isOpen ? 'block' : 'none',
        overflow: isOpen ? 'auto' : 'hidden'
      }}
      {...dataAttrs}
      {...rest}
    >
      <div lk-component="column">
        {children}
      </div>
    </div>
  );
}

```

### CSS (Reference)

```css
[lk-component="menu-list"] {
  position: absolute;
  width: var(--lk-size-3xl);
  padding: var(--lk-size-md);
  border-radius: var(--lk-size-md);
  box-shadow: var(--shadow-xl);
  transform-origin: top right;
  transition: transform 0.2s ease, opacity 0.2s ease;
}

[lk-component="menu-list"] > [lk-component="column"] {
  align-items: stretch;
  gap: calc(var(--lk-size-lg) * var(--lk-halfstep-dec));
}

[lk-menu-list-open="true"] {
  transform: scale(1);
  opacity: 1;
  overflow: auto;
  display: block;
}

[lk-menu-list-open="false"] {
  transform: scale(0);
  opacity: 0;
  overflow: hidden;
  display: none;
}

```

## Combine with Native HTML Attributes

You can set any other *native* HTML attribute (className, style, id, etc.) on the component by passing it as a prop.

## Props

### **isOpen**

**Description**

Controls whether the dropdown is shown or hidden.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `boolean` | `false` | `true`, `false` |

```tsx
<MenuList isOpen={true}>...</MenuList>

```

### **maxHeight**

**Description**

Sets the maximum height of the dropdown. Overflow becomes scrollable if content exceeds this value.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `string` | `var(--lk-size-4xl)` | Any CSS length or LK size var |

```tsx
<MenuList maxHeight="var(--lk-size-3xl)" />

```

Slots

### **children**

**Description**

The list of items or components inside the menu.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `ReactNode` | none | Any valid content |

```tsx
<MenuList>
  <MenuItem>Settings</MenuItem>
</MenuList>

```
