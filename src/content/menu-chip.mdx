# Menu Chip

## Base Component

**Default Behavior**

The `MenuChip` component is a stylized `div` used to trigger a `MenuList`. It behaves similarly to a dropdown trigger but does not rely on a `<select>` or `<input>` field under the hoodâ€”it's simply a clickable `div` with a consistent visual treatment. The component includes an icon (default: `keyboard_arrow_down`) and a `state-layer`. Visual style is inherited from the `TextInput` and `DropdownChip` base styles.

### HTML

```html
<div
  lk-component="menu-chip"
  lk-menu-chip-open="false"
>
  <span>Menu Label</span>
  <div lk-dropdown-el="chip">
    <i lk-component="icon">keyboard_arrow_down</i>
  </div>
  <div lk-component="state-layer"></div>
</div>

```

### TypeScript

```tsx
import { useMemo } from 'react';
import { propsToDataAttrs } from './utilities';

interface LkMenuChipProps extends React.HTMLAttributes<HTMLDivElement> {
  isActive?: boolean;
  label?: string;
}

export default function MenuChip({ isActive = false, label = 'Menu Label', ...rest }: LkMenuChipProps) {
  const dataAttrs = useMemo(() => propsToDataAttrs({ open: isActive }, 'lk-menu-chip'), [isActive]);

  return (
    <div {...dataAttrs} {...rest}>
      <span>{label}</span>
      <div lk-dropdown-el="chip">
        <i lk-component="icon">keyboard_arrow_down</i>
      </div>
      <div lk-component="state-layer"></div>
    </div>
  );
}

```

## Combine with Native HTML Attributes

You can set any other *native* HTML attribute (className, style, id, etc.) on the component by passing it as a prop.

## Props

### **isActive**

**Description**

Controls whether the chip is shown in its "open" visual state.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `boolean` | `false` | `true`, `false` |

```tsx
<MenuChip isActive={true} />

```

### **label**

**Description**

Sets the text label displayed inside the chip.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `string` | `Menu Label` | Any string |

```tsx
<MenuChip label="Options" />

```
