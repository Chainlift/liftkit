# Heading

## Base Component

**Default Behavior**

A `Heading` is a semantic heading element (h1 through h6) whose HTML tag is determined by the `tag` prop. The visible text is controlled via the `content` prop. Typography is controlled via the `fontClass` prop using LiftKit's global font tokens.

### HTML

```html
<h2 lk-component="heading" class="lk-font-display2-bold">Heading</h2>

```

### TypeScript

```tsx
import { useMemo } from 'react';
import { propsToDataAttrs } from './utilities';

type LkHeadingTag = 'h1' | 'h2' | 'h3' | 'h4' | 'h5' | 'h6';

interface LkHeadingProps extends React.HTMLAttributes<HTMLHeadingElement> {
  tag?: LkHeadingTag;
  fontClass?: string; // Should be LkFontClass in production
  content?: string;
}

export default function Heading({ tag = 'h2', fontClass = 'display2-bold', content = 'Heading', ...rest }: LkHeadingProps) {
  const headingAttrs = useMemo(() => propsToDataAttrs(rest, 'lk-heading'), [rest]);
  const Tag = tag;

  return <Tag className={`lk-font-${fontClass}`} {...headingAttrs}>{content}</Tag>;
}

```

### CSS (Reference)

```css
[lk-component="heading"] {
  display: block;
  margin: 0;
}

```

## Combine with Native HTML Attributes

You can set any other *native* HTML attribute (className, style, id, etc.) on the component by passing it as a prop.

## Props

### **tag**

**Description**

Controls the HTML tag used for the heading.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `LkHeadingTag` | `h2` | `h1`, `h2`, `h3`, `h4`, `h5`, `h6` |

```tsx
<Heading tag="h1" />

```

### **fontClass**

**Description**

Controls the visual style of the heading via LiftKit font tokens.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `LkFontClass` | `display2-bold` | Any LiftKit font class |

```tsx
<Heading fontClass="title1-bold" />

```

### **content**

**Description**

Sets the text content of the heading element.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `string` | `Heading` | Any string |

```tsx
<Heading content="Welcome to LiftKit" />

```
