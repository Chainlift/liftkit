# NavBar

## Base Component

**Default Behavior**

The `NavBar` is a compound component made entirely of other LiftKit primitives like `Row`, `Image`, and `IconButton`. It is always sticky, remaining at the top of the screen with `z-index: 99`. It contains layout-specific slots for different navigation groups, and supports both desktop and mobile navigation experiences.

At the **tablet breakpoint**, the `[lk-navbar-el="nav-menu"]` becomes a full-screen, fixed-position navigation drawer that slides in from the left when the user taps the menu icon.

### HTML

```tsx
<div lk-component="navbar">
  <div lk-navbar-el="navbar-content">
    <Row>
      <IconButton />
      <Link href="/">
        <Image />
      </Link>
    </Row>

    <Row lk-navbar-el="nav-menu">
      <Image />
      <Row>
        <Row lk-slot="nav-buttons">
          {navButtons}
        </Row>
        <Row lk-slot="nav-dropdowns">
          {navDropdowns}
        </Row>
      </Row>

      <Row>
        <Row lk-slot="nav-icon-buttons">
          {iconButtons}
        </Row>
        <Row lk-slot="nav-cta-buttons">
          {ctaButtons}
        </Row>
      </Row>
    </Row>
  </div>
  <div lk-navbar-el="color-scrim" />
  <div lk-navbar-el="glass-scrim" />
</div>

```

### TypeScript

```tsx
import { useMemo } from 'react';
import { propsToDataAttrs } from './utilities';

type LkMaterial = 'flat' | 'glass' | 'rubber';

interface LkNavBarProps extends React.HTMLAttributes<HTMLDivElement> {
  material?: LkMaterial;
  children?: React.ReactNode;
}

export default function NavBar({ material = 'flat', children, ...rest }: LkNavBarProps) {
  const dataAttrs = useMemo(() => propsToDataAttrs({ material }, 'lk-navbar'), [material]);

  return (
    <div {...dataAttrs} {...rest}>
      {children}
    </div>
  );
}

```

### CSS (Reference)

```css
[lk-component="navbar"] {
  position: sticky;
  top: 0;
  z-index: 99;
}

[lk-navbar-el="nav-menu"] {
  transition: transform 0.3s ease, opacity 0.2s ease;
}

@media (max-width: 768px) {
  [lk-navbar-el="nav-menu"] {
    position: fixed;
    top: 0;
    left: 0;
    height: 100svh;
    width: 100%;
    transform: translateX(-100%);
    opacity: 0;
    z-index: 98;
  }

  [lk-navbar-el="nav-menu"].active {
    transform: translateX(0);
    opacity: 1;
  }
}

```

## Combine with Native HTML Attributes

You can set any other *native* HTML attribute (className, style, id, etc.) on the component by passing it as a prop.

## Props

### **material**

**Description**

Controls the background and blur behavior of the navbar. Functions the same as the `material` prop on the `Card` component.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `LkMaterial` | `flat` | `flat`, `glass`, `rubber` |

```tsx
<NavBar material="glass" />

```
