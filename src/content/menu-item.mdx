# Menu Item

## Base Component

**Default Behavior**

The `MenuItem` component is an inline-flex `div` used inside lists, dropdowns, or menus. It optionally includes a start icon and/or an end icon and always includes a state layer. Its padding adjusts based on the presence of either icon, ensuring appropriate spacing and visual alignment.

Top and bottom padding: `var(--lk-size-lg) * var(--lk-halfstep-dec)`

Left and right padding:

- If icon present: `var(--lk-size-lg) * var(--lk-halfstep-dec)`
- If icon absent: `var(--body-font-size)`

Border radius is also `var(--lk-size-lg) * var(--lk-halfstep-dec)`.

### HTML

```html
<div lk-component="menu-item">
  <i lk-component="icon" lk-icon-position="start">home</i>
  <span>Menu Label</span>
  <i lk-component="icon" lk-icon-position="end">chevron_right</i>
  <div lk-component="state-layer"></div>
</div>

```

### TypeScript

```tsx
import { useMemo } from 'react';
import { propsToDataAttrs } from './utilities';

interface LkMenuItemProps extends React.HTMLAttributes<HTMLDivElement> {
  startIcon?: string;
  endIcon?: string;
  children?: React.ReactNode;
}

export default function MenuItem({ startIcon, endIcon, children, ...rest }: LkMenuItemProps) {
  const dataAttrs = useMemo(() => propsToDataAttrs(rest, 'lk-menu-item'), [rest]);

  return (
    <div {...dataAttrs}>
      {startIcon && <i lk-component="icon" lk-icon-position="start">{startIcon}</i>}
      <span>{children}</span>
      {endIcon && <i lk-component="icon" lk-icon-position="end">{endIcon}</i>}
      <div lk-component="state-layer"></div>
    </div>
  );
}

```

### CSS (Reference)

```css
[lk-component="menu-item"] {
  display: inline-flex;
  align-items: center;
  gap: var(--lk-size-sm);
  border-radius: calc(var(--lk-size-lg) * var(--lk-halfstep-dec));
  padding-top: calc(var(--lk-size-lg) * var(--lk-halfstep-dec));
  padding-bottom: calc(var(--lk-size-lg) * var(--lk-halfstep-dec));
}

/* Left padding with start icon */
[lk-component="menu-item"]:has([lk-icon-position="start"]) {
  padding-left: calc(var(--lk-size-lg) * var(--lk-halfstep-dec));
}

/* Right padding with end icon */
[lk-component="menu-item"]:has([lk-icon-position="end"]) {
  padding-right: calc(var(--lk-size-lg) * var(--lk-halfstep-dec));
}

/* Default padding without icons */
[lk-component="menu-item"]:not(:has([lk-icon-position="start"])) {
  padding-left: var(--body-font-size);
}

[lk-component="menu-item"]:not(:has([lk-icon-position="end"])) {
  padding-right: var(--body-font-size);
}

```

## Combine with Native HTML Attributes

You can set any other *native* HTML attribute (className, style, id, etc.) on the component by passing it as a prop.

## Props

### **startIcon**

**Description**

An optional icon displayed at the start of the item.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `string` | none | Any Material Symbol |

```tsx
<MenuItem startIcon="home">Item</MenuItem>

```

### **endIcon**

**Description**

An optional icon displayed at the end of the item.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `string` | none | Any Material Symbol |

```tsx
<MenuItem endIcon="chevron_right">Item</MenuItem>

```

### **children**

**Description**

The text or content inside the menu item.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `ReactNode` | none | Any renderable content |

```tsx
<MenuItem>Settings</MenuItem>
```
