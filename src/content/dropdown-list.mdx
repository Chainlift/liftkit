# Dropdown List

## Base Component

**Dropdown List**

The DropdownList is a `div` container that wraps a `Column` component containing one or more `MenuItem` components. It is styled with padding and scroll behaviors to create a collapsible list.

### HTML

```
<div lk-component="dropdown-list" lk-dropdown-list-is-open="false">
  <div lk-component="column">
    <div lk-component="menu-item">Option 1</div>
    <div lk-component="menu-item">Option 2</div>
  </div>
</div>
```

### TypeScript

```
import { useMemo } from 'react';
import { propsToDataAttrs } from './utilities';

interface LkDropdownListProps extends React.HTMLAttributes<HTMLDivElement> {
  isOpen?: boolean;
  children?: React.ReactNode;
}

export default function DropdownList(props: LkDropdownListProps) {
  const { children, ...restProps } = props;

  const listAttrs = useMemo(
    () => propsToDataAttrs(restProps, 'lk-dropdown-list'),
    [restProps.isOpen]
  );

  return (
    <div {...listAttrs}>
      <div lk-component="column">{children}</div>
    </div>
  );
}
```

### CSS (Reference)

```
[lk-component="dropdown-list"] {
  padding: var(--lk-size-md);
  box-sizing: content-box;
  overflow: hidden;
  max-height: 0;
  transition: max-height 0.2s ease-out;
}

[lk-dropdown-list-is-open="true"] {
  max-height: var(--lk-size-3xl);
  overflow-y: auto;
}

[lk-component="dropdown-list"] > [lk-component="column"] {
  display: flex;
  flex-direction: column;
  gap: calc(var(--body-line-height) * var(--lk-wholestep-dec));
}
```

## Combine with Native HTML Attributes

You can set any other *native* HTML attribute (className, style, id, etc.) on the component by passing it as a prop.

## Props

### **isOpen**

**Description**

Controls the visibility of the dropdown list using a collapsing max-height effect. When true, the list expands and becomes scrollable. When false, it collapses and hides overflow.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `boolean` | `false` | `true`, `false` |

```
<DropdownList isOpen={true} />
```
