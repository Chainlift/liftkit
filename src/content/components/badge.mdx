# Badge

## Base Component

**Default Behavior**

The badge is used as a sort of rich status indicator. It’s a `<div>` with an icon in it. By default, it is a flex box that hugs its contents with padding on each side equal to `var(--lk-size-xs)` and a border radius of `var(--lk-size-sm)`.

### HTML

```html
<div lk-component="badge">
  <span lk-component="icon" lk-icon-name="info"></span>
</div>
```

### TypeScript

```tsx
import { useMemo } from "react";
import { propsToDataAttrs } from "../utilities";

interface LkBadgeProps extends React.HTMLAttributes<HTMLDivElement> {
  scale?: "md" | "lg";
  color?: keyof LkColor;
  icon?: React.ReactNode;
}

export default function Badge(props: LkBadgeProps) {
  const { icon, children, ...restProps } = props;

  const lkBadgeAttrs = useMemo(() => propsToDataAttrs(restProps, "lk-badge"), [restProps.scale, restProps.color]);

  return (
    <div {...lkBadgeAttrs}>
      <div lk-component="slot" lk-slot="icon">
        {icon}
      </div>
      {children}
    </div>
  );
}
```

## Combine with Native HTML Attributes

You can set any other _native_ HTML attribute (className, style, id, etc.) on the component by passing it as a prop.

## Props

**scale**

**Description**

Controls the font size of the badge’s container, thereby scaling the size of the whole badge (since everything’s defined in `em` units).

| **Expected Type** | **Default Value** | **Acceptable Values** |
| ----------------- | ----------------- | --------------------- |
| `LkSizeUnit`      | `md`              | `md`, `lg`            |

```tsx
<Badge scale="lg" />
```

**color**

**Description**

Controls the background color of the badge container and the text color of the contained icon. Assigns the appropriate "on-\*" color to the icon for contrast.

| **Expected Type** | **Default Value**     | **Acceptable Values** |
| ----------------- | --------------------- | --------------------- |
| `keyof LkColor`   | `lk-surfacecontainer` | Any `LkColor` key     |

```tsx
<Badge color="lk-warning" />
```

**icon**

**Description**

A React component or HTML element that appears inside the badge.

| **Expected Type** | **Default Value** | **Acceptable Values**    |
| ----------------- | ----------------- | ------------------------ |
| `React.ReactNode` | _(none)_          | Any element or component |

```tsx
<Badge icon={<Icon>info</Icon>} />
```
