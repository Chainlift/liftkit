# Paragraph

## Base Component

**Default Behavior**

A `Paragraph` is a semantic `<p>` element used for body text. It shares the same styling and API approach as the `Heading` component, but it does not include a `tag` prop since it is always rendered as a paragraph. The text style is determined by the `fontClass` prop.

### HTML

```html
<p lk-component="paragraph" class="lk-font-body">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna
  aliqua. Ut enim ad minim veniam.
</p>
```

### TypeScript

```tsx
import { useMemo } from "react";
import { propsToDataAttrs } from "../utilities";

interface LkParagraphProps extends React.HTMLAttributes<HTMLParagraphElement> {
  fontClass?: string; // Should be LkFontClass in production
  content?: string;
}

export default function Paragraph({
  fontClass = "body",
  content = "Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam.",
  ...rest
}: LkParagraphProps) {
  const paragraphAttrs = useMemo(() => propsToDataAttrs(rest, "lk-paragraph"), [rest]);

  return (
    <p className={`lk-font-${fontClass}`} {...paragraphAttrs}>
      {content}
    </p>
  );
}
```

### CSS (Reference)

```css
[lk-component="paragraph"] {
  display: block;
  margin: 0;
}
```

## Combine with Native HTML Attributes

You can set any other _native_ HTML attribute (className, style, id, etc.) on the component by passing it as a prop.

## Props

### **fontClass**

**Description**

Controls the visual style of the paragraph via LiftKit font tokens.

| **Expected Type** | **Default Value** | **Acceptable Values**  |
| ----------------- | ----------------- | ---------------------- |
| `LkFontClass`     | `body`            | Any LiftKit font class |

```tsx
<Paragraph fontClass="callout" />
```

### **content**

**Description**

Sets the text content of the paragraph element.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| ----------------- | ----------------- | --------------------- |
| `string`          | Lorem ipsumâ€¦      | Any string            |

```tsx
<Paragraph content="This is a paragraph." />
```
