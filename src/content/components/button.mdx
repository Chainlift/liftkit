# Button

## Base Component

### Default Behavior

The button is a bit unique compared to previous components. It responds to mouse interactions using a state layer. It changes color when hovered, pressed, or focused by changing the state layer’s opacity.

### HTML

```tsx
import Row from "./liftkit";

<button
  lk-component="button"
  lk-button-variant="fill"
  lk-button-start-icon="some_icon_name"
  lk-button-end-icon="some_icon_name"
  type="button"
>
  <div lk-button-content-wrap>
    <i lk-component="icon" lk-icon-position="start">
      arrow_back
    </i>
    <span lk-button-child="button-text">Button Text</span>
    <i lk-component="icon" lk-icon-position="end">
      arrow_forward
    </i>
  </div>
  <div lk-component="state-layer" lk-state-layer-color="var(--lk-primary)" />
</button>;
```

### TypeScript

```tsx
import { useMemo } from "react";
import { propsToDataAttrs } from "../utilities";

interface LkButtonProps extends React.ButtonHTMLAttributes<HTMLButtonElement> {
  label: string;
  variant?: "fill" | "outline" | "text";
  color?: keyof LkColor;
  size?: "sm" | "md" | "lg";
  material?: string;
  startIcon?: string;
  endIcon?: string;
}

export default function Button(props: LkButtonProps) {
  const { label, startIcon, endIcon, children, ...restProps } = props;

  const lkButtonAttrs = useMemo(
    () => propsToDataAttrs(restProps, "lk-button"),
    [
      restProps.variant,
      restProps.color,
      restProps.size,
      restProps.material,
      startIcon,
      endIcon,
    ],
  );

  return (
    <button {...lkButtonAttrs} type="button">
      <div lk-button-content-wrap>
        {startIcon && (
          <i lk-component="icon" lk-icon-position="start">
            {startIcon}
          </i>
        )}
        <span lk-button-child="button-text">{label}</span>
        {endIcon && (
          <i lk-component="icon" lk-icon-position="end">
            {endIcon}
          </i>
        )}
      </div>
      <div
        lk-component="state-layer"
        lk-state-layer-color="var(--lk-primary)"
      />
    </button>
  );
}
```

### Combine with Native HTML Attributes

You can set any other _native_ HTML attribute (className, style, id, etc.) on the component by passing it as a prop.

## Props

### label

**Description**

Controls the button’s text. The content of the button’s text equals the value of the label prop.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| ----------------- | ----------------- | --------------------- |
| `string`          | _(required)_      | Any string            |

```tsx
<Button label="Submit" />
```

### variant

### Description

Controls whether the button is filled, outlined, or simply text.

| **Expected Type** | **Default Value** | **Acceptable Values**     |
| ----------------- | ----------------- | ------------------------- |
| `string`          | `fill`            | `fill`, `outline`, `text` |

```tsx
<Button variant="outline" />
```

### color

### Description

Controls the color based on the variant. For `fill`, it controls the background. For `outline` and `text`, it controls text and border color, and state layer color.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| ----------------- | ----------------- | --------------------- |
| `keyof LkColor`   | `lk-primary`      | Any LkColor key       |

```tsx
<Button color="lk-tertiarycontainer" />
```

### size

### Description

Controls the button size. If `sm`, button font size is subheading. If `lg`, button font size is title3.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| ----------------- | ----------------- | --------------------- |
| `LkSizeUnit`      | `md`              | `sm`, `md`, `lg`      |

```tsx
<Button size="lg" />
```

### material

### Description

Controls miscellaneous filters.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| ----------------- | ----------------- | --------------------- |
| `string`          | _(optional)_      | Any string            |

```tsx
<Button material="frosted" />
```

### startIcon

### Description

Controls the icon on the left side. Sets the value inside the `<i>` element with `lk-icon-position="start"`.

| **Expected Type** | **Default Value** | **Acceptable Values**    |
| ----------------- | ----------------- | ------------------------ |
| `string`          | `public`          | Any Material Symbol name |

```tsx
<Button startIcon="arrow_back" />
```

### endIcon

### Description

Controls the icon on the right side. Sets the value inside the `<i>` element with `lk-icon-position="end"`.

| **Expected Type** | **Default Value** | **Acceptable Values**    |
| ----------------- | ----------------- | ------------------------ |
| `string`          | _(none)_          | Any Material Symbol name |

```tsx
<Button endIcon="arrow_forward" />
```
