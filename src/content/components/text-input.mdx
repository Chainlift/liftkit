# Text Input

## Base Component

**Default Behavior**

A text input is a simple form control composed of a label, input field, optional end icon, and optional help text. It is structured to support relative positioning so that child elements like icons or a `state-layer` can be layered on top of the field.

### HTML

```html
<div lk-component="text-input">
  <label lk-text-input-element="label" class="label">Label</label>
  <div lk-text-input-element="field-wrap">
    <input lk-text-input-element="input" type="text" name="name" id="id" />
    <div lk-component="icon" lk-text-input-element="end-icon">search</div>
    <div lk-component="state-layer"></div>
  </div>
  <div lk-text-input-caption>
    <span lk-component="icon">search</span>
    <p class="caption">Help text (optional)</p>
  </div>
</div>
```

### TypeScript

```tsx
import { useMemo } from "react";
import { propsToDataAttrs } from "../utilities";

interface LkTextInputProps extends React.InputHTMLAttributes<HTMLInputElement> {
  labelPosition?: "default" | "on-input";
  helpText?: boolean;
  placeholderText?: string;
}

export default function TextInput(props: LkTextInputProps) {
  const { children, ...restProps } = props;

  const lkTextInputAttrs = useMemo(
    () => propsToDataAttrs(restProps, "lk-text-input"),
    [restProps.labelPosition, restProps.helpText, restProps.placeholderText]
  );

  return <div {...lkTextInputAttrs}>{/* implementation omitted for brevity */}</div>;
}
```

## Combine with Native HTML Attributes

You can set any other _native_ HTML attribute (className, style, id, etc.) on the component by passing it as a prop.

## Props

### **labelPosition**

**Description**

Controls the position of the input label. By default, the label appears above the input. When set to `on-input`, the label appears as a floating placeholder and may need additional CSS to shift based on the focus state.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| ----------------- | ----------------- | --------------------- |
| `string`          | `default`         | `default`, `on-input` |

```tsx
<TextInput labelPosition="on-input" />
```

### **helpText**

**Description**

Determines whether help text is rendered beneath the field. When `false`, the help text container is omitted entirely.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| ----------------- | ----------------- | --------------------- |
| `boolean`         | `false`           | `true`, `false`       |

```tsx
<TextInput helpText={true} />
```

### **placeholderText**

**Description**

Sets the placeholder text for the input. If left blank or undefined, no placeholder text is rendered.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| ----------------- | ----------------- | --------------------- |
| `string`          | _(none)_          | Any string            |

```tsx
<TextInput placeholderText="Search here..." />
```
