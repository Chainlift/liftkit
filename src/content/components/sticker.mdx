# Sticker

## Base Component

**Default Behavior**

The `Sticker` component is a minimal, inline `div` that wraps a `Text` component. Itâ€™s typically used to display small labels or identifiers, like status tags or overlays.

The internal `Text` component uses a default `fontClass` of `'label'`.

### HTML

```html
<div lk-component="sticker">
  <div lk-slot="content">sticker</div>
</div>
```

### TypeScript

```tsx
import { useMemo } from "react";
import { propsToDataAttrs } from "../utilities";

interface LkStickerProps extends React.HTMLAttributes<HTMLDivElement> {
  content?: string;
  color?: string; // Should match allowed subset of LkColor from Button
}

export default function Sticker({ content = "sticker", color = "lk-primary", ...rest }: LkStickerProps) {
  const dataAttrs = useMemo(() => propsToDataAttrs({ color }, "lk-sticker"), [color]);

  return (
    <div {...dataAttrs} {...rest}>
      <Text fontClass="label" content={content} />
    </div>
  );
}
```

### CSS (Reference)

```css
[lk-component="sticker"] {
  display: inline-block;
  padding: var(--lk-size-xs) var(--lk-size-sm);
  background-color: var(--lk-primary);
  color: var(--lk-onprimary);
  border-radius: var(--lk-size-sm);
  font-weight: 500;
  line-height: var(--label-line-height);
  font-size: var(--label-font-size);
}
```

## Combine with Native HTML Attributes

You can set any other _native_ HTML attribute (className, style, id, etc.) on the component by passing it as a prop.

## Props

### **content**

**Description**

Controls the sticker label content.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| ----------------- | ----------------- | --------------------- |
| `string`          | `"sticker"`       | Any string            |

```tsx
<Sticker content="Beta" />
```

### **color**

**Description**

Controls the background color of the sticker, and the `on` color is automatically applied to the text.

| **Expected Type** | **Default Value** | **Acceptable Values**           |
| ----------------- | ----------------- | ------------------------------- |
| `LkColor`         | `lk-primary`      | Same subset as Button component |

```tsx
<Sticker color="lk-success" />
```
