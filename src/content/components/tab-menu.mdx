# Tab Menu

## Tab Menu

**Default Behavior**

The `TabMenu` component is a `div` containing a Row component that organizes multiple `TabLink` components. It has the following default styling:

- `display: flex`
- `flex-direction: row`
- `align-items: stretch`
- `justify-content: flex-start`
- `width: 100%`
- `overflow-x: auto`
- `padding: 0`
- `position: relative`
- `border-bottom: 1px solid var(--lk-outlinevariant)`

- props
    - tabLinks
        - a list of strings. each string is automatically turned into a tab link.
        - type is array of strings
        - default [”Tab Link 1”, “Tab Link 2”, “Tab Link 3”]
    - alignItems
        - passes the value to the Row component’s alignItems prop
        - same rules as the row’s alignItems prop
        - default is unset
    - justifyContent
        - passes the value to the Row component’s justifyContent prop
        - same rules as the row’s justifyContent prop
        - default is unset
