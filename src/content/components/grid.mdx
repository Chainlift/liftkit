# Grid

## Base Component

The grid component is simply a `div` block with the attribute `lk-component=”grid”`.

`Grid` displays with 2 columns and 1 row by default, with a base grid gap of `var(--lk-size-md)`.

### HTML

```html
<div lk-component="grid">{children}</div>
```

### TypeScript

#### Definition

```tsx
import { useMemo } from "react";
import { propsToDataAttrs } from "../utilities";

// Assuming these are defined/imported somewhere globally
type LkVariable = string;
type CssSize = string;

// The LiftkitGrid type definition
interface LkGridProps extends React.HTMLAttributes<HTMLDivElement> {
  columns: number;
  gap: LkVariabl;
  autoResponsive: boolean;
}

export default function Grid(props: LkGridProps) {
  // Extract children and any other props you don't want passed as custom attributes
  // Do this on every component.
  const { children, ...restProps } = props;

  // Memoize attribute creation to improve performance and prevent unnecessary recalculations
  // Do this for every component.
  const lkGridAttrs = useMemo(
    () => propsToDataAttrs(restProps, "lk-grid"),
    [
      restProps.columns,
      restProps.gap,
      restProps.minmax,
      restProps.minWidth,
      restProps.maxWidth,
      restProps.autoResponsive,
    ]
  );

  return <div {...lkGridAttrs}>{children}</div>;
}
```

#### Example instance

```tsx
import Grid from "./Grid";

function ExampleGridPage() {
  return (
    <Grid
      columns={4}
      gap="var(--lk-spacing-md)"
      autoResponsive={true}
      // className and style, below, exemplify how you can pass normal HTML
      // attributes in addition to the Liftkit attributes.
      className="my-custom-grid"
      style={{ margin: "2rem 0" }}
    >
      <div>Item 1</div>
      <div>Item 2</div>
      <div>Item 3</div>
      <div>Item 4</div>
    </Grid>
  );
}
```

### CSS

```css

  grid-template-columns: repeat(2, 1fr);
  grid-template-rows: auto;

```

## Combine with Native HTML Attributes

### Use Base HTML Attributes

You can set any other _native_ HTML attribute (class, id, name, etc) on the component by passing it as a prop. This is true for all components.

## Props

`Grid` accepts the following props which pass their values to their respective custom attributes.

### `columns`

Controls the number of columns in the grid.

| Expected Type | **Default Value** | **Acceptable Values**                   | Implementation         |
| ------------- | ----------------- | --------------------------------------- | ---------------------- |
| number        | 2                 | Accepts positive integers from 1 to 12. | `<Grid columns={3} />` |

### `gap`

Controls the `gap` property. Columns and rows are always symmetrical.

| **Expected Type**                | Default Value       | **Acceptable Values**    | **Implementation**                 |
| -------------------------------- | ------------------- | ------------------------ | ---------------------------------- |
| LkVariable (CSS variable string) | `var(--lk-size-md)` | Any global size variable | `<Grid gap="var(--lk-size-xs)" />` |

### `autoResponsive`

| **Expected Type** | **Default Value** | **Acceptable Values**                                                   | **Implementation**              |
| ----------------- | ----------------- | ----------------------------------------------------------------------- | ------------------------------- |
| Boolean           | false             | If true, forces grids to collapse to 1 column at the mobile breakpoint. | `<Grid autoResponsive={true}/>` |
