# Text

## Base Component

**Default Behavior**

The `Text` component is a generic block of inline-readable content rendered by default as a `<div>`. It shares the same typography system as the `Paragraph` component but allows users to override the tag with any semantic HTML element such as `<caption>`, `<article>`, etc.

### HTML

```html
<div lk-component="text" class="lk-font-display2-bold">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam.
</div>

```

### TypeScript

```tsx
import { useMemo } from 'react';
import { propsToDataAttrs } from './utilities';

type LkSemanticTag = keyof JSX.IntrinsicElements;

interface LkTextProps extends React.HTMLAttributes<HTMLElement> {
  fontClass?: string; // Should be LkFontClass
  content?: string;
  tag?: LkSemanticTag;
}

export default function Text({ tag = 'div', fontClass = 'display2-bold', content = 'Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam.', ...rest }: LkTextProps) {
  const textAttrs = useMemo(() => propsToDataAttrs(rest, 'lk-text'), [rest]);
  const Tag = tag;

  return <Tag className={`lk-font-${fontClass}`} {...textAttrs}>{content}</Tag>;
}

```

### CSS (Reference)

```css
[lk-component="text"] {
  display: block;
  margin: 0;
}

```

## Combine with Native HTML Attributes

You can set any other *native* HTML attribute (className, style, id, etc.) on the component by passing it as a prop.

## Props

### **fontClass**

**Description**

Controls the visual style of the text via LiftKit font tokens.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `LkFontClass` | `display2-bold` | Any LiftKit font class |

```tsx
<Text fontClass="body" />

```

### **content**

**Description**

Sets the text content of the rendered element.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `string` | Lorem ipsumâ€¦ | Any string |

```tsx
<Text content="Custom display text." />

```

### **tag**

**Description**

Sets the HTML tag used for rendering. This is useful for less common tags like `<caption>`, `<article>`, etc.

| **Expected Type** | **Default Value** | **Acceptable Values** |
| --- | --- | --- |
| `LkSemanticTag` | `div` | Any valid tag name |

```tsx
<Text tag="caption" />
```
